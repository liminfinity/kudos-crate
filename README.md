# Опросница

Платформа корпоративной обратной связи и аналитики команд.

## Возможности

- Обратная связь между коллегами
- Kudos — система благодарностей
- Company Mood Index — индекс настроения компании
- Critical Incidents — отслеживание серьёзных инцидентов
- Полугодовые опросы и дневник руководителя
- AI-модерация текстов (орфография, токсичность)
- AI-ассистент Мира — контекстная помощь

## Технологии

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Lovable Cloud (база данных, аутентификация, edge functions)
- Lovable AI Gateway (модерация текста, ассистент)
